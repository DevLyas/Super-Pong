## Dependencies

- C compiler (gcc, clang, ...)
- Raylib
- Cmake

## Quick start

- create the build directory and enter it  
`mkdir build && cd build`

- run cmake  
`cmake ..`

- build the game  
`make`

- run the game  
`./pong`
